# svm-loan-classifier
# **Project overview**
<br>
SVC based classification model which analyses applicant's financial details to determine if they are eligible for loan or not. 
<br>
It is important to verify and analyse the loan applicant's financial details before approving loan for them to prevent financial loss.This project builds a machine learning classifier to predict an applicant's eligibility for a loan.
<br>
By analyzing key features such as Loan_Amount_Terms and credit history, the model identifies high-risk customers, allowing businesses to take proactive retention measures.
